Linux命令- - - scp 服务器间传输，基于ssh

关于此tag，emmm，在工作学习中，接触到一些命令是必不可少的了，有时，我觉得一些命令挺有用处，就会尝试着记录下来，希望能坚持。这个是第一篇，像是文档一样，同时希望以后能更有趣的介绍Linux commands。💕

scp是 secure copy的缩写，scp是Linux系统下基于ssh登录，进行的远程文件拷贝命令

scp简易写法

scp 【可选参数】【原路径】 【目标路径】

可选参数 (常见)

  • -r：递归复制整个目录
  • -C：允许压缩
  • -p：保留原文件的修改时间，访问时间，访问权限。
  • -P port：指定数据传输用到的端口号。如 -P 8080
  • -I limit：限定用户所能使用的带宽，kbit/s 为单位
  • -v：详细方式显示输出，可以用于调试连接，验证和配置问题。
  • -i identity_file： 从指定文件中读取传输时使用的密钥文件，此参数直接传递给ssh。

示例

  1. 从本地上传到远端，（1、2需要已指定用户的密码，3、4需要输入用户名和密码）

    1. scp /www/test.txt root@192.168.1.1:/www/scp 上传到远端的scp目录下
    2. scp /www/test.txt root@192.168.1.1:/www/scp/new_test.txt 上传到远端的scp目录下并修改文件名
    3. scp /www/test.txt 192.168.1.1:/www/scp 上传到远端的scp目录下
    4. scp /www/test.txt 192.168.1.1:/www/scp 上传到远端的scp目录下并修改文件名
    5. 如果是文件夹，就加上 -r 参数，如 scp -r /www/test/ 192.168.1.1:/www/scp/
    6. PS：这里有个关于权限的问题，如果登录到远程的用户是普通用户，则不能上传到根目录，而是上传到用户目录。

  2. 从远端下载到本地

    1. 下载和上传类似，换换上面的位置即可。如 scp -r 192.168.1.1:/www/scp/ /www/test/
    2. PS：如果远程的服务器开启防火墙并指定的scp的端口号，则需要使用 -P参数来设置这个端口号，如scp -r -P 8080 192.168.1.1:/www/scp/ /www/test/
