如何在Table里面写一个Tooltip（文字提示

Element-ui有一个组件，是tooltip，作用是替代元素默认title属性。

还有一个组件是Table，其子元素有个属性是 show-overflow-tooltip，作用是，将展示省略号的内容，以鼠标移动到上面，展示全部信息。

那么，如何实现一个在表格里面，给过多的内容，显示全部内容，足够显示的，就不用了呢。

需求点剖析

  1. 鼠标移动到元素上，如果文本过长而显示…时，则出现tooltip，否则不需要出现
  2. tooltip提示框，能以元素的中点居中展示，向两端扩展

思路解析

  1. 鼠标移动，则看起来是需要监听鼠标移动事件，又考虑到众多td元素，可以给td添加鼠标事件
  2. 文本过长显示… 可以用css实现。单行文本过长显示省略号
  3. 如果只在文本过长时，才显示tooltip，则需要判断元素的文本是否过长
  4. tooltip提示框要相对目标元素居中展示，可以用绝对定位来解决。那么则需要获取到目标元素的位置信息
  5. 三角箭头可以用border实现

涉及知识

  1. 鼠标事件，有 mouseenter 和 mouseleave 事件，这两个分别是监听鼠标进入和离开目标元素，刚好符合我们的需求，并且mouseleave和mouseout是不同的，后者会事件冒泡
  2. 单行文本过长显示省略号，这个需要给元素固定宽度，然后设置overflow: hidden，超过隐藏，设置white-space: nowrap，不换行，设置text-overflow: ellipsis，超出文本以省略号显示。
  3. scrollWidth，是一个元素的可滚动宽度。尽管文本看起来是显示省略号了，但实际上它的scroll宽度还是有的，那么，可以用scrollWidth 和 clientWidth的大小来判断是否过长
  4. 元素有个方法，是获取它相对于屏幕的位置，getBoundingClientRect，返回值是个对象。包括元素的x y值，宽高值。这样，就可以用绝对定位，找到tooltip提示框的位置了。
  5. 使用border其中一个方向，填充颜色，另外三个就设置transparent。
